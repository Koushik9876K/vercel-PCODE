How To SetUp Your Blogger

# Go to your blogger Dashboard > Pages

Login to your Blogger account. On the left sidebar menu, click on the Pages tab.

# Create New page name survey (make sure select HTML View)

Click on New Page, enter "survey" as the title. Click the pencil icon and switch to HTML View.

# Copy the code below and paste it on your blogger page.
HTML Code 

              <script>
              // Get job_id and worker_id from URL parameters
              const urlParams = new URLSearchParams(window.location.search);
              const job_id = urlParams.get("job");
              const worker_id = urlParams.get("worker");
              
              // Store job_id and worker_id in localStorage
              localStorage.setItem("identity", job_id + worker_id);
          </script>
